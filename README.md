#### MODULE NAME
       MUX2_5

#### DEPENDENCIES
       MUX2

#### INPUTS
Name | Size(Bits)
-----|------------
S   |     1      
A   |     5      
B   |     5      

#### OUTPUTS
Name | Size(Bits) 
-----|------------
Y   |     5      

#### DESCRIPTION
       Two-input multiplexor with five-bit inputs.  Sets output Y equal to:

       A if S = 0,

       B if S = 1.
